Servicing Our Autos

The Servicing Our Autos project involved the creation and documentation of multiple interconnected web services. These web services implemented a business process provided to the students. Shortest Distance

Overview

Road networks are modeled utilising Graph Theory. For a set of locations and possible road segments which have been randomly generated, the route of the shortest path from a point to all other points can be found. A Minimum-Spanning Tree can also be calculated, as well as it's shortest path. This system employs the use of Dijkstra's Single Source Shortest Path algorithm, Kruskal's Minimum Cost Spanning Tree algorithm and Breadth First Search Algorithm. The final system is a command line application that can read data in from text files.

This application was developed by a student for INB371 at QUT during Semester 1, 2014.

Technologies

C++

Recommendation Evaluation

Overview

Recommendation algorithms are commonly used in e-commerce and social media websites to market relevant content to users. The Recommender Evaluation project involved developing and implementing algorithms to recommend movies based on user information, and then evaluating these recommendations.

The deliverables required to complete this project were:

  • Data pre-processing
  • Implementing four distinct recommendation algorithms
    • Item Based Collaborative Filtering
    • User Based Collaborative Filtering
    • Latent Dirichlet allocation
    • Word-based Pattern Mining
  • Producing an evaluation of the recommendations using two metrics
    • Precision: how relevant the recommendations are
    • Recall: how many relevant recommendations have been found
  • Technical report

This application was developed by five students for INB302 at QUT during Semester 2, 2014. I was one of the two programmers in the group. For showcase purposes the results were summarised using a website (see below).

Technologies

Java, MySQL, Flat Files, LensKit API
