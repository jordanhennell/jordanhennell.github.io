Overview

TripPlanner is a web application that allows people to plan holidays by adding activities from major cities in Australia. Users select the city they want to visit and then activities for that city are displayed. They can add activities to a trip for later review. If a user is an Activity Provider they can also add activities to cities.

This web application was developed by two students for INB373 at QUT during Semester 1, 2014. The applications was assessed based on functionality, not design.

Technologies

ASP.NET (C# and VB), MSSQL, IIS, HTML, CSS, JavaScript

Disaster Management

Overview

The Disaster Management web application was to bring together all disaster emergency services to assist with coordinated disaster mitigation. Users of the system were able to report new incidents (eg. flood, fire, etc.), while also finding locations of existing incidents. The application also allows coordinaters to organise their personnel by deploying them to incidents.

This web application was developed by six students for INB201 at QUT during Semester 1, 2013.

Technologies

ASP.NET (C#), MSSQL, IIS, HTML, CSS, JavaScript, Google Maps
