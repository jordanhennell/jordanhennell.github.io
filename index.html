Overview

TripPlanner is a service that allows people to register and plan holidays by adding activities that are linked to major cities in Australia. When users sign-up, they can choose whether they want to be able to add activities to the system as well as being able to plan trips. Once registered, users begin by entering the Trip Planning page and clicking New Trip. Once pressed, the cities are dynamically displayed from the database as link buttons. Users select the city they want to visit and then activities bound to that city are displayed, dynamically from the database. Users select with activities they want to add to their trip, and these are written to and stored in a trip text file in their user directory. Once finished they click End Trip and from there they can view the trip with all the activities in from their Saved Activities page. If a user is an activity provider, they can add activities to cities through the Create Activity page. Users input information into the fields and then this is added to the database. If a user wants to change their password then they can do so through the Account page. This is also displays if the user is an administrator for TripPlanner or not.
